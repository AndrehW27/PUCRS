Objeto por fábricas


    
        function criarPessoa(nome, idade) {
            return {
                nome,
                idade,
                falar() {
                    console.log(`Meu nome é ${nome} e tenho ${idade} anos.`);
                }
            };
        }
        const pessoa1 = criarPessoa("Maria", 30);
        const pessoa2 = criarPessoa("Carlos", 40);
    

    


    
Objeto por construtor
 
    

    
        function Pessoa(nome, idade) {
            this.nome = nome;
            this.idade = idade;
            this.falar = function() {
                console.log(`Meu nome é ${this.nome} e tenho ${this.idade} anos.`);
            };
        }
        
        const pessoa1 = new Pessoa("Ana", 28);
    
 
    
 

    
Objeto por protótipos
 
    

    
        const prototipoPessoa = {
            falar() {
                console.log(`Meu nome é ${this.nome}`);
            }
        };
        
        const pessoa = Object.create(prototipoPessoa);
        pessoa.nome = "Pedro";

Conceito de encapsulamento

        

            O encapsulamento é o princípio de esconder os detalhes internos de um objeto e expor apenas uma interface controlada. Em JavaScript, isso pode ser feito usando closures ou propriedades privadas
        

    
Conceito de abstração

        

            A abstração permite modelar apenas os aspectos relevantes de um objeto, ocultando detalhes desnecessários. Em JavaScript, isso é feito ao expor apenas métodos e propriedades essenciais.
        

    
Conceito de herança
  
        

            A herança permite que uma classe herde atributos e métodos de outra classe, promovendo a reutilização de código. Em JavaScript, a herança é implementada usando a palavra-chave extends.
        
 
    
Conceito de polimorfismo

        

            O polimorfismo permite que objetos de diferentes classes sejam tratados de forma uniforme, desde que compartilhem uma interface comum. Em JavaScript, isso é natural devido à tipagem dinâmica.
